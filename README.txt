== Description ==

Personal WP is a clean, elegant and professional Wordpress theme. It is thought for freelancers which need an easy and straightforward online presence. Such as web designers, physiotherapists and personal trainers. The theme has a clean UX/UI and a clean code which can be easy twisted according to your needs. Through media query and CSS grid the theme is fully responsive. New sections are coming soon! This is a great starting point to launch your successful career.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Select Personal WP theme
3. Click on the 'Activate' button to use your new theme right away.
